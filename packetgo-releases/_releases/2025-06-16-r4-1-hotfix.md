## 개요

컨트롤러 및 게이트웨이에 대한 최적화 및 추가 개선 사항이 반영된 핫픽스 패키지입니다.

## 업데이트 내용

### 1. 컨트롤러 표준 제품 최적화 적용

사용자 접속 지연 및 DB 부하 완화를 위한 최적화 적용

### 2. 이중화 구성 기능 개선

- 이중화 구성 시 콘솔 입력 방식(대화형) 지원
- 이중화 구성 시 `keepalived.conf` 설정 파일 내 `virtual_router_id` 값을 **사용자 지정**이 가능하도록 개선
- 가이드: 패키지관리 > PRIBIT Connect 2.6 R4.1 > 문서

### 3. 보안 코딩 기준 적용

NSR 신뢰성 검증 과정에서 요구된 시큐어 코딩 기준을 적용하여 보안성 강화

### 4. DBMS 스키마 변경

- DBMS 스키마 명칭 변경 (`DB_RISK` → `DB_PGZT`)
  - 기존 설치 시: `DB_RISK` 유지
  - 신규 설치 시: `DB_PGZT`로 생성
  - 업데이트 시: 설치/업데이트 패키지에서 기존 스키마를 조회하여 자동 반영
  - 이중화 구성 시: DB 스키마 조회 후 자동 반영되도록 수정

### 5. 3rd-party 구성 요소 최신 버전 적용

`install_offline.sh` 기준:

| 구성 요소 | 버전 |
|----------|------|
| OpenSSL | v3.0.16 |
| Nginx | v1.26.3 |
| OpenJDK | v17.0.14 |
| Embedded Tomcat | v10.1.34 |
| MariaDB | v10.6.21 |
| Redis | v7.4.2 |
| Keepalived | v2.0.19 |
| Snmpd | v5.8 |
| bwm-ng | v0.6.2 |

> `install_online.sh`는 인터넷 연결을 통해 저장소의 최신 버전으로 설치됨

### 6. 컨트롤러 일본어 제품 언어 팩 적용

## 참고 사항

- 성능 최적화에 관련된 `my.cnf` 설정 내용은 **PRIBIT Connect 2.6 R4.1 활동 기록**을 참고하여 진행
- Ubuntu 24.04의 경우 R5에서 진행 예정
