## 주요 업데이트 내역

### 1. MacOS, iOS, iPadOS, Android, Android Tablet Agent 기능 지원

- 데이터 플로우 2차 인증 기능 구현
- 생체 인증 설정 기능
- 간편 비밀번호
- 간편 비밀번호 재설정 기능
- 모바일 QR 만 사용 시 터널 없이 진행되게 구현
- 사용자 정보에 따른 안내 팝업 다수 구현

### 2. 고정 IP 할당 기능 개선

- 고정 IP 설정을 ID별이 아닌 접속 단말별로 설정 가능하도록 개발
- 고정 IP 모니터링 현황에서 고정 IP(IP-Sec / SSL)을 단말 별로 현황 볼 수 있도록 기능 개선
- SSL, IPSec 우선순위를 네트워크 경계에서 각 사용자별 VPN 정보로 이관

### 3. 설치 패키지 업데이트

- 우분투 자동 업데이트 기능 비활성화 처리 (스크립트 수정 및 관련 서비스 비활성화)
- mariadb 공식 repository의 mirror 사이트 변경
- 게이트웨이 업데이트 시 ssh 등 서비스 관련 설정 파일 변경되지 않도록 수정

### 4. OpenVPN 다운그레이드 (v2.6.10 → v2.5.4)

메모리 Leak 증상 있어, 이전에 사용한 버전으로 다운그레이드 진행 (현재 수정 중)

### 5. VPN 인증 정보 동기화 기능 개선

SSL VPN과 IP-Sec VPN 인증 정보가 동시에 동기화되도록 개선

### 6. SSL VPN 계정 정보 처리 개선

- SSL VPN 계정 정보 저장 방식 변경: 계정 정보 파일저장 → Redis(Memory DB)에 저장
- VPN 접속 인증 처리 방식 변경: 스크립트 인증처리 → Gateway RPC호출로 인증처리

### 7. PC 에이전트 인터페이스 종료버튼 추가

PC 에이전트 모든 화면에서 종료(x)버튼 클릭 시 종료 팝업 노출되도록 인터페이스 개선

### 8. 인증코드 알림 언어 설정 개선

에이전트에서 사용하는 언어에 맞게 인증코드 문구 발송되도록 개선
