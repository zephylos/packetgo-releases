## 개요

보안 취약점 패치 및 iOS 26 / macOS 26 호환성 대응을 포함한 핫픽스 패치입니다.

## 주요 변경 사항

### 1. 보안 취약점 패치

다음 취약점들이 수정되었습니다.

- 회사 메일 계정 및 비밀번호, SMTP 설정 정보 노출 취약점
- 컨트롤러 / 게이트웨이 내 불필요한 서비스 정보 노출 취약점
- 직접 명령어 실행을 통한 VPN 터널 생성 가능 취약점
- Windows 에이전트를 우회하여 VPN 터널 생성 가능 취약점
- Windows 메모리 내 중요 정보 노출 취약점
- Windows 라우팅 테이블 조작으로 보안 터널 우회 가능 취약점
- Windows TAP-Adapter 설정 변경으로 보안 터널 무력화 가능 취약점
- 콘솔 웹 페이지 내 관리자 서비스 접근 통제 우회로 인한 권한 상승 취약점
- 콘솔 웹 페이지 내 컨트롤러 아이디 중복 설정 방지 기능 우회 취약점

### 2. iOS 26 / macOS 26 호환성 지원

- iOS 26 및 macOS 26 환경에서의 동작 호환성 검증 수행 후 필요 수정 적용
- macOS 26 (Tahoe) 호환성 검증 적용 완료
