## 개요

PacketGo ZTNA 2.6 최초 릴리즈입니다. 총 673건 중 수정된 529건이 반영되었습니다.

## 테스트 환경

| 항목 | 내용 |
|------|------|
| OS | ubuntu-20.04.6, Linux Kernel 5.4.0-144, x86-64 |
| DB | MariaDB 10.6.17 |
| Agent | Windows 11 64Bit |

## 주요 업데이트 내역

### 1. DashBoard 메뉴 개발

- Usage Matrics: 사용자 및 단말 OS 정보 등
- Rules: 정책 및 차단 내역 등
- Resources: 컨트롤러, 게이트웨이 성능 분석 (리스크 분석 내용 이관)

### 2. TCP 세션 인증 기능 개발

- Flow Routing 대역에 해당하는 Flow에 대해서만 DataFlow ID 체크 및 TCP SYN 인증 기능을 수행
- Flow Routing 외의 대역에 대해서는 Bypass 시키도록 구현
- Flow Routing 대역에는 있고 목적지 설정이 되어 있지 않은 IP는 Redirection 되도록 구현

### 3. UDP 패킷제어 기능 개발

- 플로우 라우팅 대역 내 UDP 패킷을 제어
- 플로우 라우팅 대역 외 대역은 제어하지 않음
- DNS, DHCP 제외

### 4. 대용량 로그 조회 기능 개발

시스템 환경 설정에 로그 보관 기간 설정 추가

### 5. 인사연동 기능 추가

API 기반으로 사용자 계정 정보 동기화 방식 추가

### 6. 공인 IP 정보 제공

에이전트 실행 시 해당하는 공인 IP를 서버에 제공하는 형식

### 7. 에이전트 인증 화면에서 컨트롤러 시스템 환경 설정에 영향을 받지 않게 개선

사용자 인증 완료 시각이 사용자 MFA 만료 시각보다 늦을 경우 인증 시간이 남아도 대기 만료가 됐던 현상 수정

### 8. 서로 다른 OS에서 같은 계정으로 에이전트 동시 접속 가능하게 개선

- SSL VPN에서 동일한 인증 정보 접속 시 기존 정보를 삭제하는 문제 해결
- 제이플로우 아이디를 이용하여 VPN 인증 정보가 겹치지 않도록 수정

### 9. SMS 인증 개발

강화된 인증 SMS 방식 추가

### 10. OpenVPN Windows tap드라이버 이름 변경

`tap0901.sys` → `tap2560.sys` (드라이버 이름 충돌 방지)

### 11. 설치 패키지 업그레이드

- OpenSSL: 3.2.1 → 3.0.13
- MariaDB: 10.6.16 → 10.6.17
- java 17, tomcat 10, nginx 1.24.0

### 12. java 외부 라이브러리 업데이트

spring-boot 관련 라이브러리: 3.2.2 → 3.2.4
