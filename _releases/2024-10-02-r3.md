## 주요 업데이트 내역

### 1. PRIBIT Connect ZTNA 2.6 UI 개선

- POLICY 정책 통합
- 이전 페이지 기능 추가
- 일괄 등록, 상태 변경, 삭제 기능 추가
- 애플리케이션 플로우 프로세스 ANY 추가

자세한 ZTNA 2.6 UI개선 사항은 아래 링크의 문서 참조:
[ZTNA 2.6 UI 개선 문서](https://pribit.daouoffice.com/app/works/applet/18737/doc/3097780/navigate/0)

### 2. OpenSSL 업데이트

v3.0.13 → v3.0.14

### 3. OpenVPN 업데이트

v2.5.4 → v2.6.12

### 4. MFA 2차 인증 시 TOTP 연동 기능 개발

Microsoft Authenticator 앱 또는 Google Authenticator 앱 OTP 연동 기능 개발

### 5. SSL VPN 전송 계층 프로토콜 옵션 설정 기능 개발

- 네트워크 경계 SSL VPN 설정 시 전송 프로토콜 설정 가능하도록 기능 구현
- `server.conf` UDP 프로토콜 사용 옵션 추가

### 6. VPN 동시 사용자 접속을 위한 설정 변경

`max-clients` 250 → 10000으로 수정

### 7. 안드로이드 VPN 권한 요청 시점 수정

사용자 로그인 후 터널 접속 시도 시, VPN 권한 팝업 노출

### 8. 게이트웨이 로그 활성화/비활성화 기능 추가

`config.ini` 파일에서 로그 활성화 및 비활성화를 설정할 수 있도록 기능 수정

### 9. 게이트웨이 최초 설치 시 네트워크 장애 처리 기능 개선

컨트롤러와 연동 전에 네트워크 사용할 수 없는 상태 발생하여 해당 기능 개선

### 10. 단일IP 단위 허용 개수 수정

기존 80개 등록에서 200개 이상 등록 허용

### 11. 사용자 계정 동기화 데이터베이스 추가

Oracle, MSSQL, PostgreSQL 사용자 계정 동기화 기능 추가

### 12. 간편 비밀번호 초기화 문구 변경

간편 비밀번호 재설정 → 간편 비밀번호 초기화로 문구 변경

### 13. 컨트롤러 다국어 영문 번역 수정 반영
