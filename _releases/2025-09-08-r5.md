## 개요

Ubuntu 24.04 공식 지원, Gateway 오프라인 설치 지원, macOS 에이전트 Overwrite 업데이트 등 주요 개선 사항이 포함된 정규 릴리즈입니다.

## 주요 업데이트 내역

### 1. Ubuntu 24.04.02 지원

- Controller
  - 20.04 설치 파일: `PRIBIT-Connect-Controller-Focal_2.6.x.x.deb`
  - 24.04 설치 파일: `PRIBIT-Connect-Controller-Noble_2.6.x.x.deb`
  - 업데이트 파일: `PRIBIT-Connect-Controller-Updater_2.6.x.x.deb`
- Gateway: 기존 동일 (`PRIBIT-Connect-Gateway_2.6.x.x.deb`)

### 2. Gateway 오프라인 설치 지원

오프라인/온라인 설치 및 업데이트 옵션 제공.

- 오프라인 설치: `./start-wizard.sh --off_line`
- 오프라인 업데이트: `./start-wizard.sh --off_line --update`
- 온라인 설치: `./start-wizard.sh`
- 온라인 업데이트: `./start-wizard.sh --update`

### 3. macOS 에이전트 패키지 Overwrite 업데이트 지원

기존: 에이전트 삭제 후 설치 필요 → 개선: 설치 시 기존 에이전트를 Overwrite하여 설치

### 4. macOS 네트워크 경계 기반 라우팅 할당 기능

- 기존: `server.conf` 설정 대역 기반으로 라우팅 할당
- 개선: 네트워크 경계 대역 > 에이전트에 할당된 IP 대역으로 라우팅 할당

### 5. 컨트롤러 정책 복사 기능 추가

애플리케이션 플로우, 단말 상태 확인 및 행위 제어 메뉴 내 정책 복제 기능 추가

### 6. 컨트롤러 사용자 검색 기능 개선

- 기존: 이름만 검색/노출/선택 가능
- 개선: 이름(아이디) 검색/노출/선택 가능

### 7. 가상 IP 그룹 관리 메뉴 표준 적용

고객사 전용 메뉴 표준 패키지 적용, 그룹 및 사용자별 VPN IP 할당 관리 가능

### 8. Windows 재배포 패키지 포함

설치 패키지에 재배포 패키지를 포함

### 9. OS별 로그인 로직 정의

일반 로그인, 자동 로그인, 간편 로그인 로직 정의 (참고: Figma > 정책 및 UI정의 > 로그인 로직 정의)

### 10. 인증 번호 제공 방법 개선

- 기존: Email 또는 SMS 단일 방식
- 개선: Email / SMS / Email+SMS (사용자가 인증 방식 선택 가능)

### 11. 데이터 플로우 아이디 관리 로직 추가

Controller 서비스 구동 시 DB/Redis 간 데이터 플로우 ID 동기화 로직 추가

### 12. 안드로이드 패킷 필터 제거 버전 개발

## 제약 사항

- Windows 에이전트: VPN 직접 연결 취약점 보완으로 인해 릴리즈 제외 (추후 별도 릴리즈 예정)
